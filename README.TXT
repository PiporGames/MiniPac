MiniPac es un proyecto de C++ que consiste en un programa que calcula la mejor ruta posible para guiar un <<Pacman>> a lo largo de un laberinto bidimensional hasta su salida, evitando por el camino a los <<Fantasmas>> que quieren acabar con el.
El proyecto utiliza diversos algoritmos de cálculo de caminos, unos más "inteligentes" que otros desde algoritmos sencillos voraces, pasando por backtracking y algoritmos A*.
El objetivo del proyecto es, evidentemente, familiarizarse y comprender estos tipos de algoritmos y saber utilizarlos para un propósito específico... En este caso, salvar a nuestro amigo en forma de pizza.
Existe documentación disponible para el proyecto en el repositorio.
Este proyecto ha sido creado para la asignatura de Inteligencia Artificial y Sistemas Inteligentes del Grado de Ingeniería en Ingeniería del Software de la UEx año 2023/2024.  

Creado por José Manuel de Torres Domínguez y otros autores acreditados en la documentación del proyecto.

En este repositorio se encuentra una serie de archivos C++ compilables.
Para su ejecución es recomendable utilizar el script de compilación de bash para Linux.
Es necesario tener las herramientas de compilación como Make para su compilación, además de las indicadas en la documentación.
